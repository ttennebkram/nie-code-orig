Managing Directed Results Help:
Searching by Term or URL

This screen lets you locate Directed Results Maps that you would like to Edit or Delete.

The Search Term box lets you search by term.

The URL box lets you look for Maps that point to specific URLs.

You can use one or more asterisks (*) as wildcard characters in either box. This is particularly helpful when searching for URLs. For example you could search for *faq* instead of http://mycompany.com/support/faq17.html.

You cannot search for a term and URL at the same time.

To see ALL of the Directed Results Maps, leave all the boxes empty and click the Go button.
